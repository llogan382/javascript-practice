What is GraphQL?

Is it is a query language

It is strictly typed.

It can only do two things:

  1. It can QUERY DATA
  2. It can MUTATE DATA

MUTATIONSAlso include changing, updating, and deleting the data

QUERIESmake is easy to create RELATIONSHIPS wiht the data

GRAPHQL has no logic- sorting, filtering, etc.

It is just a way to get what you want

Pass this info to the SERVER, where it filters, sorts, etc.

GraphQL is just a STANDARD to fetch the data.
